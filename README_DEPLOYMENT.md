# Essentia Environments - Updated Website

This project contains all pages, components, and styling for the Essentia Environments website clone built with Next.js.

## Pages Created
- /projects - Projects overview
- /projects/residential - Residential projects
- /projects/commercial - Commercial projects
- /services - Services page
- /career - Career/jobs page
- /elevation-design - Elevation design services
- /landscape-designing - Landscape design services
- /turnkey-execution - Turnkey execution services
- /furniture - Bespoke furniture
- /furniture/indoor - Indoor furniture
- /furniture/outdoor - Outdoor furniture
- /privacy - Privacy policy
- /terms - Terms & conditions
- /returns - Return policy

## Build & Deploy
Build command: `npm run build`
Start command: `npm start`
